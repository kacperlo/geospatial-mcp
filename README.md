# MCP OSM DuckDB Server

MCP Server (Model Context Protocol) for working with geospatial data using OpenStreetMap, Nominatim and DuckDB Spatial.

## Functionalities

- **Geocoding** - changing the address/place name for lat/long (Nominatim API)
- **Fetching data from OSM** - elastic requests to Overpass API with cache in DuckDB
- **Spatial SQL queries** - data analysis with DuckDB Spatial and spatial SQL queries
